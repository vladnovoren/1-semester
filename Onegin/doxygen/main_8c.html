Описывает общий алгоритм программы
Открываются файлы для чтения (f) и записи (fout) в бинарном режиме по путям, переданным в коммандную строку.
Выделяется буфер buff, в который мы передаем текст с помощью функции Read_Buff.
В массиве указателей index по их адресам посредством функции Make_Index собирается информация о строках файла.
С помощью qsort мы сортируем строки, передавая параметром нужный компаратор Str_Comp_Begin либо Str_Comp_End С помощью функций Output_Sorted_Text и Output_Original_Text выводим сортированный либо несортированный тексты
